مرحله ۳ — IP های سالم Cloudflare
ℹ
برای یافتن IP های سالم Cloudflare می‌توانید از ابزار CF Clean IP Scanner استفاده کنید. این ابزار مبتنی بر Termux بوده و روی اندروید اجرا می‌شود — نتیجه اسکن را مستقیم در کادر زیر وارد کنید.

هر تعداد IP قابل قبول است — از هر IP چند کانفیگ با پورت‌ها و متدهای مختلف ساخته می‌شود. علاوه بر IP، می‌توانید از برخی دامنه‌ها نیز استفاده کنید (مثلاً chatgpt.com)

در صورت فعال بودن، تنظیمات fragment فقط در کانفیگ مادر JSON اعمال می‌شود و روی کانفیگ‌های VLESS تأثیری ندارد.

Local DNS برای resolve دامنه‌های ایرانی استفاده می‌شود. Remote DNS به‌صورت پیشفرض روی Cloudflare DoH تنظیم است و می‌توانید آن را با هر آدرس DoH دلخواه جایگزین کنید. با ابزار DoH Proxy می‌توانید یک سرور DoH اختصاصی روی Cloudflare بسازید و آدرس آن را اینجا وارد کنید. Allow LAN اتصال سایر دستگاه‌های شبکه محلی را فعال می‌کند.

ℹ
با انتخاب بیشتر، کانفیگ‌های متنوع‌تری برای دور زدن فیلترینگ هوشمند ساخته می‌شه. هر ترکیب پورت + fingerprint یک کانفیگ مجزاست. پیشنهاد: همه‌ را تست کن و بهترین‌ها را نگه دار.

اگر خالی بماند: بدون Fragment نام «Least Ping Normal 🚀» و با Fragment نام «Least Ping Fragment 🚀» خواهد بود.
